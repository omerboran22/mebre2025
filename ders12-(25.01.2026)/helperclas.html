Helper Classes in CSS

Helper classes are utility classes that can be used to apply specific styles to elements without the need for custom CSS for each element. They are often used to quickly adjust layout, spacing, typography, and other visual aspects of a webpage.

Common Helper Classes

  • .text-center: Centers text within an element.
  • .margin-top-10: Adds a top margin of 10 pixels.
  • .padding-20: Adds padding of 20 pixels on all sides.
  • .bg-light: Applies a light background color.
  • .d-flex: Applies flexbox display to an element.
